Dashboard Data Model

The complete data structure powering all 5 variations. Follows the feedback hierarchy: Trends → Score → Focus → Strengths.

1. Treatment Progress

• Days completed / total
• Per-day status (perfect/partial/missed)
• Treatment % complete
• Days remaining
• Current week/phase

2. Today's Score

• Composite score (0–100)
• Time completed %
• Light positioning %
• Consistency %
• Session timing accuracy %

3. Trends

• Daily score history (sparkline)
• Week-over-week change %
• Trajectory (improving/stable/declining)
• Streak count (consecutive days)
• Per-metric directional change

4. Focus & Strengths

• Lowest-scoring metrics
• Actionable recommendation
• Pattern recognition (day/time)
• Highest-scoring metrics
• Streak & consistency wins

5. Patterns & Coaching

• Weekly heatmap (day-of-week × week) — identifies tough days
• Time-of-day correlation — when do best sessions happen?
• One focused coaching nudge — specific, data-backed, actionable (e.g., "try before 9 AM")
• SuperBetter-aligned: small daily wins build momentum; track physical/behavioral/environmental resilience
